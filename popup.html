vinput

Statements
nmap/xmap/omap/imap/map/map! KEY CMD1 CMD2 ...

Map KEY to execute one or more commands.

nmap, xmap, omap, and imap bind the key in normal, visual, operator-pending, and insert mode respectively. map binds the key in all but insert mode, and map! binds the key in every mode.

KEY must have the form [A-][C-][S-]KEYNAME, where KEYNAME follows the KeyboardEvent.key specification. For more details on valid key names, see the default configuration, or the w3 specification.

nmap/xmap/imap KEY operator CMD1 CMD2 ...

Map KEY to execute a new operator. Before executing the operator, the user must perform a motion. This motion will select a region of text, which the operator must then operate on using the commands specified.
